# ①課題番号-プロダクト名

RSSフィード保存アプリ

## ②課題内容（どんな作品か）

- ユーザが登録したRSSフィードから最新記事が更新されたかどうかを監視する
- 更新があれば、その記事をFireBaseのRealtimeDatabase上に保存する

- CloudFunctionを使ってみたかったが、有料だったので使えなかった
- ユーザ認証やFireStore、通知など他にも使いたい機能は多くあったが使えなかった
